PROJECTS

Openstack Swift Layer-C
Developed from June 2015 to Present. Supervised by Dr.Zhenhua LI.
Project source code repository: [Python version - DEPRECATED] [Golang version]

Openstack Swift Layer-C is a midware for Openstack Swift, implementing for maintaining robust filesystem index in Swift and providing atomic file / folder modification operation. It is done by committing and merging patches in Distributed Segment Tree. A patent about the algorithm is submitted and being reviewed.

Js2Jv (Javascript-to-Java Translator, Javascript Interpreter)
Developed from May 2015 to July 2015, by Leiyu ZHAO, Junlin LIU.
