Openstack Swift Layer-C
Developed from June 2015 to Present. Supervised by Dr.Zhenhua LI and Dr.Yunhao LIU.
Project source code repository: [Python version - DEPRECATED] [Golang version]

Openstack Swift Layer-C is a midware for Openstack Swift, implemented for maintaining robust filesystem index in Swift and providing atomic file / folder modification operation. It is done by committing and merging patches in Distributed Segment Tree. A patent about the algorithm is submitted and being reviewed.

Paper Recommendation System
Developed from April 2014 to June 2014, by Leiyu ZHAO and Junlin LIU.

The system works on a paper database to recommend contents to potential readers. By using hybrid algorithm (combining collaborative filtering and Tf-Idf) and dynamic estimation, the recommendation accuracy is estimated to be more than 71% (AP@5). A technical report describing the algorithm in detail is provided.

Panorama Photostitch
Developed from July 2013 to September 2013, by Leiyu ZHAO (Group leader), Yang YUAN, Jun QIAN.

The tool stitches several related photos to one panorama. By matching SURF vector of images, calculating transforming matrices, searching optimum stitch using minimal-cut algorithm and weakening color discrepancies, photos can be stitched together naturally.
